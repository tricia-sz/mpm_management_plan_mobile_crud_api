# Mobile Plan Management CRUD API

## Ambiente

 - Node 22.12.0
 - Docker 27.5.1

## Instalação

 - npm install  
 - docker-compose up --build
 - npx prisma migrate dev --name init
 
 ## Iniciando aplicação

 - npm run start:dev

 ## Comandos úteis

 - npx prisma studio

 ## Acessando Swagger

 - http://localhost:3000/doc
